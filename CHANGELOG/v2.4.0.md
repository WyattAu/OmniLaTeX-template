# Changelog v2.4.0 -- 2026-06-06

## Testing & Code Quality

### Added

- **buildlib unit tests**: 90 new tests across 6 files (config, ui, runner, diff, builder, cli)
- **buildlib commands tests**: 30 new tests covering preflight, scaffold, init, doctor, check, diff, lint, watch
- **buildlib property tests**: 14 property-based tests for hash determinism, log parsing, command runner
- **buildlib integration tests**: 12 integration tests for scaffold-institution, scaffold-language, init
- **SOURCE_DATE_EPOCH validation**: Tests verifying reproducible build environment propagation
- **Pre-commit hooks**: Updated to include buildlib tests in fast suite

### Fixed

- **CommandRunner timeout bug**: readline loop blocked forever, preventing process.wait(timeout) from firing. Replaced with daemon reader thread.
- **check_semver.py**: Fixed f-string bug (plain strings `{name}` instead of f-strings `{name}`)
- **buildlib coverage**: Increased from 0% to 47% (commands.py: 4% -> 54%)

## CI/CD Security

### Fixed

- **Secret leakage via echo**: Replaced inline `echo "${{ secrets.* }}"` with env binding in docker-digest-sync.yml and _docker-setup.yml
- **Script injection via matrix**: Pass matrix.example via env var instead of shell expansion in build-examples.yml
- **Overly broad permissions**: Moved contents:write to job level in dependabot-auto-merge.yml, removed workflow-level id-token:write from docker-ci.yml
- **Cache inefficiency**: Removed --force flag from visual-regression build step that invalidated the PDF cache

## UI/UX Accessibility

### Added

- **ARIA tablist semantics**: Category tabs in PDF gallery now have role="tab", aria-selected, aria-controls
- **Lightbox dialog**: Added role="dialog", aria-modal="true", hidden attribute, focus trapping, focus restoration
- **Label associations**: Fixed for="lang" and for="inst" on gallery filter selects
- **aria-current="page"**: Added to all active navigation links
- **aria-live="polite"**: Added to dynamically updated document grid
- **prefers-reduced-motion**: Added media query to shared CSS
- **Print styles**: Added @media print stylesheet (hide nav/footer, show link URLs)
- **Favicon**: Added SVG favicon to all HTML pages
- **og:image**: Added Open Graph image meta tags to all pages
- **theme-color**: Added theme-color meta tag to all pages

### Fixed

- **Broken gallery-app.html reference**: Fixed in mkdocs.yml and docs/index.md

## Security Headers

### Added

- **HSTS**: Strict-Transport-Security with max-age=63072000, includeSubDomains, preload
- **Permissions-Policy**: camera=(), microphone=(), geolocation=(), interest-cohort=()
- **Global security headers**: Applied to all routes (/*) instead of HTML only

- **CSS extraction**: Extracted inline CSS from 3 HTML pages to external stylesheets (index.css, gallery.css, verify.css)
