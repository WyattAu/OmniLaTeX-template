# OmniLaTeX v2.1.0

## Added

### Phase 2 Test Coverage Expansion

- Module compilation tests for 5 previously untested lib/ areas:
  - `lib/layout/` (page, KOMA, floats, boxes)
  - `lib/typography/` (math notation, delimiters, siunitx)
  - `lib/references/` (hyperref, cross-references)
  - `lib/language/` (i18n, translations)
  - `lib/tables/` (booktabs, tabularray)
- All 9 module compilation tests now pass (was 4)
- Beamer regression tests: colors, KOMA skips, compatibility stubs, native doctype
- CLI tests for 7 additional subcommands: doctor, lint, check, clean, export, diff
- Total test count: 753 passed, 52 skipped (was 761/52 before module test restructure)

### TEXINPUTS Resolution

- `run_module_tests.py` now sets TEXINPUTS with correct priority order
  (module test dir first, then lib/config/lua/repo root)
- Prevents main.tex/main.aux conflicts when compiling in subdirectories

## Fixed

- Removed `--halt-on-error` from all module test `.latexmkrc` files
  (consistent with root `.latexmkrc` which avoids it for multi-pass builds)
- Fixed fragile command expansion in `\typeout` (OmniBuildMode, MintedDraftState)
- Added required color palette and translation stubs to module test fixtures
- CTAN script test now skips conditional `cp` targets (guarded by `if [ -f ]`)
- Graphics module test now loads `omnilatex-base` (provides `\ifOmniUltra`)
- Glossary module test uses manual translations (avoids polyglossia/datatool CJK errors)
