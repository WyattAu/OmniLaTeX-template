# Changelog

## [1.22.0] - 2026-05-11

### Added

- **Docker digest sync automation** -- `docker-digest-sync.yml` now updates all 5 downstream CI configs (GitLab, Gitea, Forgejo, Woodpecker, devcontainer) alongside `.env.docker`
- **Visual regression CI workflow** -- `.github/workflows/visual-regression.yml` with build, diff, and regenerate jobs; triggered on `.tex/.sty/.cls/.lua` changes
- **`--regenerate-references` flag** -- `build.py diff` supports copying built PDFs to `tests/references/` as new baselines
- **`tests/references/.gitkeep`** -- directory for visual regression reference PDFs
- **6 doctype l3build regression tests** -- thesis, article, manual, presentation, cv, book (placeholder .tlg files for l3build check --generate)

### Changed

- **CHANGELOG split** -- monolithic 580-line CHANGELOG.md replaced with index + 23 per-version files in `CHANGELOG/`
- **README.md dedup** -- doctype table replaced with 3-row base class summary; build.py commands reduced to 4 essentials; links to USER_GUIDE for details
- **USER_GUIDE.md dedup** -- Quick Start and Project Structure sections replaced with cross-references to README

### Verified

- Docker digest audit: all platforms use identical sha256 digest (automated sync prevents future drift)
- README/USER_GUIDE: ~70 lines of duplicated content eliminated
- 375 fast tests pass (48 skipped, 1 expected xfail)
