% English abstract
\section*{Abstract}
This thesis investigates cross-lingual transfer learning techniques for
low-resource natural language processing.  We propose a novel method that
leverages multilingual pre-trained models to improve performance on
languages with limited labeled data.

\foreignlanguage{german}{\section*{Zusammenfassung}
Diese Thesis untersucht Cross-Lingual Transfer Learning für
ressourcenarme NLP.  Wir schlagen eine neue Methode vor, die
mehrsprachige vortrainierte Modelle nutzt, um die Leistung auf
Sprachen mit begrenzten beschrifteten Daten zu verbessern.}

\foreignlanguage{french}{\section*{Résumé}
Cette thèse étudie les techniques d'apprentissage transversal pour
le traitement du langage naturel à faibles ressources.  Nous
proposons une nouvelle méthode qui exploite les modèles multilingues
prédéfinis pour améliorer les performances sur les langues avec
des données étiquetées limitées.}
