# EPFL (École Polytechnique Fédérale de Lausanne) Thesis Example

A thesis template demonstrating EPFL (École Polytechnique Fédérale de Lausanne) institutional branding with OmniLaTeX.

## Description

This example shows a complete thesis setup with EPFL (École Polytechnique Fédérale de Lausanne) configuration, featuring:

- Institutional colors, title page layout, and page headers/footers
- Front matter, main matter, and back matter organization
- Mathematical typesetting with equations and tables
- Bibliography management with BibLaTeX

## Building

```bash
python build.py build-example thesis-epfl
```

## Customization

Replace `institution=epfl` with any other institution or `none` for a generic template. See `config/institutions/` for all available institutions.
