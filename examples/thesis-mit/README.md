# Massachusetts Institute of Technology (MIT) Thesis Example

A thesis template demonstrating Massachusetts Institute of Technology (MIT) institutional branding with OmniLaTeX.

## Description

This example shows a complete thesis setup with Massachusetts Institute of Technology (MIT) configuration, featuring:

- Institutional colors, title page layout, and page headers/footers
- Front matter, main matter, and back matter organization
- Mathematical typesetting with equations and tables
- Bibliography management with BibLaTeX

## Building

```bash
python build.py build-example thesis-mit
```

## Customization

Replace `institution=mit` with any other institution or `none` for a generic template. See `config/institutions/` for all available institutions.
