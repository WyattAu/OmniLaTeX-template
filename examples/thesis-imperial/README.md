# Imperial College London Thesis Example

A thesis template demonstrating Imperial College London institutional branding with OmniLaTeX.

## Description

This example shows a complete thesis setup with Imperial College London configuration, featuring:

- Institutional colors, title page layout, and page headers/footers
- Front matter, main matter, and back matter organization
- Mathematical typesetting with equations and tables
- Bibliography management with BibLaTeX

## Building

```bash
python build.py build-example thesis-imperial
```

## Customization

Replace `institution=imperial` with any other institution or `none` for a generic template. See `config/institutions/` for all available institutions.
