\chapter{Document Type Reference}
\label{ch:doctypes}

This appendix provides a consolidated reference of all 26 canonical
doctypes, their underlying KOMA-Script class, category, user-facing
aliases, and key configuration parameters.  See Chapter~\ref{ch:doctype-system}
for the full architectural description.

\section{Example \texttt{\textbackslash documentclass} Lines}

The following examples show the minimal class invocation for common
doctype selections:

\begin{minted}{latex}
% Academic documents
\documentclass[doctype=article]{../../omnilatex}
\documentclass[doctype=journal]{../../omnilatex}
\documentclass[doctype=technicalreport]{../../omnilatex}
\documentclass[doctype=thesis, institution=tuhh]{../../omnilatex}
\documentclass[doctype=dissertation]{../../omnilatex}
\documentclass[doctype=research-proposal]{../../omnilatex}

% Business documents
\documentclass[doctype=cv]{../../omnilatex}
\documentclass[doctype=cover-letter]{../../omnilatex}
\documentclass[doctype=memo]{../../omnilatex}
\documentclass[doctype=invoice]{../../omnilatex}
\documentclass[doctype=white-paper]{../../omnilatex}

% Education documents
\documentclass[doctype=homework]{../../omnilatex}
\documentclass[doctype=exam]{../../omnilatex}
\documentclass[doctype=lecture-notes]{../../omnilatex}
\documentclass[doctype=syllabus]{../../omnilatex}
\documentclass[doctype=handout]{../../omnilatex}

% Visual documents
\documentclass[doctype=poster]{../../omnilatex}
\documentclass[doctype=presentation]{../../omnilatex}

% Reference documents
\documentclass[doctype=manual]{../../omnilatex}
\documentclass[doctype=book]{../../omnilatex}
\documentclass[doctype=standard]{../../omnilatex}
\documentclass[doctype=patent]{../../omnilatex}
\documentclass[doctype=dictionary]{../../omnilatex}
\end{minted}

The doctype resolution pipeline normalises the input string to lowercase
before matching against the alias lists.  Unrecognised doctypes fall back
to \texttt{book} with a warning in the log.  Each canonical doctype has a
corresponding profile file in \texttt{config/document-types/} that defines
layout, spacing, font mode, and title page behaviour.
