\chapter{Translation Key Reference}

OmniLaTeX uses \pkg{translator} to manage localised strings for all 18
supported languages.  Every caption, heading, and label is controlled by a
\emph{translation key}.  This chapter lists all 47 keys and explains how to
add new ones.

\section{Adding New Keys}

To introduce a new translation key:

\begin{enumerate}
  \item Declare the key with an English default:
        \begin{verbatim}
\DeclareTranslation{english}{MyNewKey}{My Value}
        \end{verbatim}
  \item Add the same key to all 18 supported languages.  Each language file
        in \texttt{tex/latex/omnilatex-lang/} needs an entry:
        \begin{verbatim}
\DeclareTranslation{german}{MyNewKey}{Mein Wert}
\DeclareTranslation{french}{MyNewKey}{Ma Valeur}
\DeclareTranslation{japanese}{MyNewKey}{私の値}
% ... and so on for all languages
        \end{verbatim}
  \item Use the key in your templates or documents with
        \verb|\translate{MyNewKey}|.
\end{enumerate}

All translation declarations must appear in the preamble.  If a key is
missing for a given language, \pkg{translator} falls back to English
automatically.
