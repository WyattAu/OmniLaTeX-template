\chapter{Module Reference}
\label{ch:modules}

OmniLaTeX's functionality is distributed across 27 library modules
organised under \texttt{lib/}.  Each module is a self-contained
\texttt{.sty} file loaded conditionally by the main class or by other
modules.  The table below lists every module with its category and
purpose.

  lib/core/omnilatex-base

  lib/typography/omnilatex-fonts
    & Typography

Module loading follows a dependency graph enforced in the main class
file.  The \texttt{enable*} boolean class options gate modules at the
top level: \texttt{enablemath}, \texttt{enabletikz},
\texttt{enableengineering}, \texttt{enablecode}, and
\texttt{enabletables}.
