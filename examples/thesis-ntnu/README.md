# Norwegian University of Science and Technology Thesis Example

A thesis template demonstrating Norwegian University of Science and Technology institutional branding with OmniLaTeX.

## Description

This example shows a complete thesis setup with Norwegian University of Science and Technology configuration, featuring:

- Institutional colors, title page layout, and page headers/footers
- Front matter, main matter, and back matter organization
- Mathematical typesetting with equations and tables
- Bibliography management with BibLaTeX

## Building

```bash
python build.py build-example thesis-ntnu
```

## Customization

Replace `institution=ntnu` with any other institution or `none` for a generic template. See `config/institutions/` for all available institutions.
