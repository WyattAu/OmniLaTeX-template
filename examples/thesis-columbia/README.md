# Columbia University Thesis Example

A thesis template demonstrating Columbia University institutional branding with OmniLaTeX.

## Description

This example shows a complete thesis setup with Columbia University configuration, featuring:

- Institutional colors, title page layout, and page headers/footers
- Front matter, main matter, and back matter organization
- Mathematical typesetting with equations and tables
- Bibliography management with BibLaTeX

## Building

```bash
python build.py build-example thesis-columbia
```

## Customization

Replace `institution=columbia` with any other institution or `none` for a generic template. See `config/institutions/` for all available institutions.
