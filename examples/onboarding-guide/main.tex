\section*{Welcome to NovaPulse}

Congratulations on joining NovaPulse Technologies. This guide will walk you
through your first week, introduce you to the people you will work with most
closely, and help you get your tools and environment set up so you can start
contributing from day one.

Your onboarding buddy is \textbf{Jamie Rivera} (Senior Software Engineer).
Jamie will be your go-to resource for questions about team culture, tools,
and day-to-day life at NovaPulse. Reach out to Jamie anytime on Slack at
\texttt{@jamie.rivera}.

\section{Tools \& Accounts Setup}

\subsection{Day 1 Checklist}

\begin{itemize}
    \item[\(\square\)] Receive laptop from IT (MacBook Pro M3 Max or
          ThinkPad X1 Carbon, depending on preference).
    \item[\(\square\)] Verify laptop is enrolled in MDM (Jamf or Intune).
    \item[\(\square\)] Install 1Password and join the NovaPulse vault.
    \item[\(\square\)] Create company Google Workspace account
          (\texttt{yourname@novapulse.ai}).
    \item[\(\square\)] Join Slack workspace and key channels:
          \texttt{\#general}, \texttt{\#engineering}, \texttt{\#team-engineering},
          \texttt{\#incidents}, \texttt{\#random}.
    \item[\(\square\)] Set up VPN access (WireGuard configuration provided
          by IT).
    \item[\(\square\)] Verify Workday access and complete profile.
\end{itemize}

\subsection{Day 2 Checklist}

\begin{itemize}
    \item[\(\square\)] Create GitHub account (if personal) and join the
          NovaPulse organization.
    \item[\(\square\)] Accept GitHub invite and verify repository access.
    \item[\(\square\)] Install development tools: VS Code (or preferred IDE),
          Docker Desktop, Python 3.11+, Node.js 20+, PostgreSQL client.
    \item[\(\square\)] Clone the main repositories:
          \texttt{novapulse/platform}, \texttt{novapulse/edge-sdk},
          \texttt{novapulse/ml-pipeline}.
    \item[\(\square\)] Follow the \texttt{CONTRIBUTING.md} guide to set up
          your local development environment.
    \item[\(\square\)] Run the full test suite locally and confirm all tests
          pass.
\end{itemize}

\subsection{Day 3 Checklist}

\begin{itemize}
    \item[\(\square\)] Set up Linear account and join the
          \texttt{NovaPulse Engineering} workspace.
    \item[\(\square\)] Familiarize yourself with the team's board structure
          and sprint workflow.
    \item[\(\square\)] Install \texttt{gh} CLI tool and authenticate with
          GitHub.
    \item[\(\square\)] Set up local PostgreSQL database and seed it with
          test data.
    \item[\(\square\)] Complete the ``Hello World'' edge deployment tutorial
          using the NovaPulse SDK.
\end{itemize}

\subsection{Day 4 Checklist}

\begin{itemize}
    \item[\(\square\)] Review and bookmark the engineering handbook

    \item[\(\square\)] Set up Grafana dashboard access (request from DevOps).
    \item[\(\square\)] Configure PagerDuty mobile app for on-call rotation

    \item[\(\square\)] Review the incident response runbook

    \item[\(\square\)] Complete all remaining IT security training modules.
\end{itemize}

\subsection{Day 5 Checklist}

\begin{itemize}
    \item[\(\square\)] Pick up your first starter task from Linear.
    \item[\(\square\)] Submit your first pull request (even if it is small).
    \item[\(\square\)] Get your PR reviewed by at least one team member.
    \item[\(\square\)] Complete the onboarding feedback survey in Workday.
    \item[\(\square\)] Schedule your 30-day check-in with your manager.
\end{itemize}

\section{30-60-90 Day Plan}

\subsection{Days 1--30: Learn}

\begin{itemize}
    \item Complete onboarding checklist and tool setup.
    \item Shadow at least 3 customer calls or demos.
    \item Submit 5+ pull requests (starter tasks and small features).
    \item Read all architecture decision records (ADRs) in the repository.
    \item Build a relationship with your onboarding buddy and manager.
    \item Attend at least one sprint retrospective and provide feedback.
\end{itemize}

\subsection{Days 31--60: Contribute}

\begin{itemize}
    \item Own a small feature or improvement end-to-end.
    \item Participate in code reviews for teammates' pull requests.
    \item Write or update documentation for a component you worked on.
    \item Join the on-call rotation (shadowing first, then primary).
    \item Present a short ``tech talk'' or demo to the team on something
          you learned.
    \item Complete all mandatory compliance training.
\end{itemize}

\subsection{Days 61--90: Lead}

\begin{itemize}
    \item Take ownership of a well-scoped project or initiative.
    \item Mentor a newer team member or incoming intern.
    \item Contribute to sprint planning by estimating and proposing work.
    \item Identify one process improvement and propose it to the team.
    \item Complete your 90-day performance conversation with your manager.
    \item Celebrate your first quarter at NovaPulse.
\end{itemize}

\section{Frequently Asked Questions}

\begin{description}
    \item[How do I request time off?] Submit through Workday under the
        ``Time Off'' tab. Your manager will be notified and can approve or
        discuss. For unplanned absences, notify your manager via Slack as
        early as possible.
    \item[Who do I contact for IT issues?] Post in \texttt{\#help-it} on
        Slack or email \href{mailto:it@novapulse.ai}{it@novapulse.ai}.
        Urgent issues (locked out, security incident): call the IT hotline
        at +1 (555) 012-3456.
    \item[What if I have an accessibility need?] Contact People Operations
        at \href{mailto:people@novapulse.ai}{people@novapulse.ai}. We will
        work with you to provide appropriate accommodations.
    \item[Can I work from a co-working space?] Yes. NovaPulse provides a
        WeWork All Access membership. Activate through the Ramp card or
        request reimbursement for other providers.
    \item[When is my first paycheck?] Payroll runs bi-weekly on Fridays.
        You will receive your first paycheck on your second Friday of
        employment.
    \item[How do I update my personal information?] Log in to Workday and
        navigate to ``Personal Information'' to update your address, bank
        accounts, tax withholding, and emergency contacts.
\end{description}
