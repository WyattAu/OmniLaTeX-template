\section{Introduction}
This section outlines the scope of the article~\cite{bibid}.  The central
question motivating this work can be stated as follows: given a dataset
$\mathcal{D} = \{(\mathbf{x}_i, y_i)\}_{i=1}^{N}$, how should one choose a
model class $\mathcal{M}$ that balances expressiveness with generalisation?
We address this question through both theoretical analysis and empirical
evaluation.

\section{Methodology}
Our approach builds on the framework of regularised empirical risk
minimisation.  Given a hypothesis $h \in \mathcal{H}$, the objective function
is
\begin{equation}
    \mathcal{L}(h) = \frac{1}{N} \sum_{i=1}^{N} \ell\bigl(h(\mathbf{x}_i),\; y_i\bigr)
    + \lambda\, \Omega(h),
    \label{eq:objective}
\end{equation}
where $\ell$ is a convex loss function and $\Omega(h)$ is a complexity penalty
with regularisation strength $\lambda > 0$.

\section{Discussion}
The results in Table~\ref{tab:benchmarks} demonstrate consistent improvements
across all three benchmarks~\cite{einstein1905}.  The largest gain is observed
on Dataset B, which contains the highest label noise, suggesting that the
regularisation term effectively mitigates overfitting.
