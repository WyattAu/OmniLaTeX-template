\section{Selected Projects}

\subsection{Kinetic Typography Engine}

\textbf{Role:} Lead Developer \hfill \textbf{Year:} 2025

An open-source web tool that generates animated typography from text input. Users type a phrase and the engine produces real-time kinetic typography animations with customizable motion paths, easing functions, and color palettes.

\begin{itemize}
    \item Built with React, Three.js, and WebGPU for GPU-accelerated rendering.
    \item 4,200+ GitHub stars; featured on Hacker News front page.
    \item Used by design students at 12 universities for motion typography coursework.
\end{itemize}

\subsection{Urban Soundscape Map}

\textbf{Role:} Creative Technologist \hfill \textbf{Year:} 2024

An interactive data visualization mapping the acoustic environments of 200 European cities. Each neighborhood is rendered as a sonified color field that plays a generative soundscape when clicked, derived from real audio recordings and environmental data.

\begin{itemize}
    \item Commissioned by the Haus der Kulturen der Welt (HKW) for their ``Sonic Futures'' exhibition.
    \item Processed over 50,000 hours of ambient audio data.
    \item Built with D3.js, Web Audio API, and Mapbox GL.
\end{itemize}

\subsection{Accessible Health Dashboard}

\textbf{Role:} Full-Stack Developer \hfill \textbf{Year:} 2024

A patient-facing health monitoring dashboard for a telemedicine startup. The platform visualizes vitals (heart rate, blood pressure, glucose levels) with real-time updates and generates weekly summaries for physician review.

\begin{itemize}
    \item WCAG 2.1 AA compliant with screen reader optimization and keyboard navigation.
    \item Served 12,000+ patients across 3 German health insurance providers.
    \item Reduced appointment no-shows by 23\% through automated reminders.
\end{itemize}

\subsection{Procedural City Generator}

\textbf{Role:} Solo Developer \hfill \textbf{Year:} 2023

A real-time procedural city generation system using wave function collapse (WFC) and L-systems. The tool generates endless, explorable urban environments with consistent architecture and street layouts.

\begin{itemize}
    \item Rendered in real time at 60 FPS on consumer hardware.
    \item Presented at procedural generation workshop at SIGGRAPH Asia 2023.
    \item Open-source release with 1,800+ downloads in the first month.
\end{itemize}
