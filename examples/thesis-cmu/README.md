# Carnegie Mellon University (CMU) Thesis Example

A thesis template demonstrating Carnegie Mellon University (CMU) institutional branding with OmniLaTeX.

## Description

This example shows a complete thesis setup with Carnegie Mellon University (CMU) configuration, featuring:

- Institutional colors, title page layout, and page headers/footers
- Front matter, main matter, and back matter organization
- Mathematical typesetting with equations and tables
- Bibliography management with BibLaTeX

## Building

```bash
python build.py build-example thesis-cmu
```

## Customization

Replace `institution=cmu` with any other institution or `none` for a generic template. See `config/institutions/` for all available institutions.
