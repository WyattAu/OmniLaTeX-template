# Contributing an Institution Configuration

This guide explains how to add a new institution configuration to OmniLaTeX.

## Prerequisites

- A logo file (PDF or PNG, square aspect ratio recommended)
- Knowledge of the institution's brand colors and typography

## Steps

### 1. Scaffold the Configuration

Run the build script to create a new institution from the generic template:

    python build.py scaffold-institution <institution-name>

This creates `config/institutions/<institution-name>/` with:
- `<institution-name>.sty` -- the configuration file
- `assets/logos/<institution-name>/` -- directory for logo files

### 2. Customize the Style File

Edit `config/institutions/<institution-name>/<institution-name>.sty`:

Required fields:
| Field | Macro | Description |
|-------|-------|-------------|
| Package declaration | `\ProvidesPackage{...}` | Auto-generated by scaffold |
| Primary color | `OmniInstitutionColor` | Main brand color |
| Secondary color | `OmniInstitutionAccentColor` | Accent/secondary color |
| Logo | `\LogoInstitution` | Path to logo file |
| Name translations | `\DeclareTranslation{...}{LogoInstitution}{...}` | At least for `english` |

### 3. Add Logo Files

Place logo files in `assets/logos/<institution-name>/`:
- `logo.pdf` -- vector logo for PDF output
- `logo.png` -- raster logo for HTML preview (optional)

### 4. Validate

Run the test suite to verify structural integrity:

    python -m pytest tests/test_institutions.py -v

### 5. Test Compilation

Build an example with the new institution:

    python build.py build-example thesis

With `doctype=thesis,institution=<institution-name>` in `main.tex`.

### 6. Commit

Follow the contribution guidelines in CONTRIBUTING.md.

## Naming Conventions

- Use lowercase, hyphenated names: `my-university` (not `MyUniversity`)
- The directory name, .sty filename, and logo directory must match exactly
- Use the institution's common English short name

## Checklist

- [ ] `config/institutions/<name>/<name>.sty` exists
- [ ] `\ProvidesPackage` declaration is present
- [ ] `OmniInstitutionColor` is defined (hex color)
- [ ] `OmniInstitutionAccentColor` is defined (hex color)
- [ ] Logo file exists in `assets/logos/<name>/`
- [ ] `tests/test_institutions.py` passes for the new institution
- [ ] At least one example compiles with `institution=<name>`
